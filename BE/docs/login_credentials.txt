# Contoh Login SPK Job Portal

Untuk login, gunakan format berikut:

**Email:** `test_user@example.com`  
**Password:** `dummy123`

**Cara pakai:**
1. Kirim POST request ke **/api/auth/login**
2. Sertakan header: `Content-Type: application/json`
3. Body (JSON):
```json
{
  "email": "test_user@example.com",
  "password": "dummy123"
}
```

**Contoh curl:**
```bash
curl -X POST http://localhost:3000/api/auth/login \\
  -H "Content-Type: application/json" \\
  -d '{"email": "test_user@example.com", "password": "dummy123"}'
```

> 💡 **Catatan:** Sistem sudah memiliki data uji (test users) di database. Jika ingin login dengan akun nyata, gunakan email dan password yang terdaftar di sistem.
